# 🏥 HealthLens Analytics

## 📊 Overview
HealthLens Analytics is an interactive healthcare dashboard developed using Power BI. This project analyzes healthcare data and transforms it into meaningful insights through dynamic visualizations and KPI metrics. The dashboard helps users monitor patient health trends, demographics, and key medical indicators.

---

## 🎯 Objectives
- Analyze healthcare data effectively
- Monitor patient health metrics
- Identify trends and patterns
- Support data-driven decision making

---

## 🛠️ Tools Used
- Power BI
- Microsoft Excel

---

## 📈 Key Performance Indicators (KPIs)

| KPI | Description |
|------|------------|
| 👥 Total Patients | Total number of patients analyzed |
| 📊 Average Age | Average age of patients |
| ⚖️ Average BMI | Average Body Mass Index |
| ❤️ Average Blood Pressure | Average BP of patients |
| 🩸 Average Cholesterol | Average cholesterol level |
| 🚨 High Risk Patients | Patients with critical health conditions |
| 👨 Male Patients | Total male patients |
| 👩 Female Patients | Total female patients |
| 🏥 Disease Count | Total diseases identified |
